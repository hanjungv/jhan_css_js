속성 주기

저 속성 그대로 사용하는 경우는 거의 없을 것(안 이쁘기 때문에)

1. input type = 'text', 'email', 'password' : 그냥 있는대로 속성이 잘 먹음, 없앨 것도 크게 없음. validation을 자동으로 해준다는 장점


2. input type = 'date' : 달력을 선택해 주는 input, 하지만 IE, firefox, safari 지원을 안한다. 모바일은 걍 밀고가고(사실 더 편한 방법이 있는지 잘 모르겠..)웹에선 jquery date picker



3. textarea : 이것도 말은 잘 듣지만 resize가 가능한 상태가 default이므로 수정해주자



4. input type = 'file' : 말 드럽게 안듣는다. 사람들이 원하는 모양도 각자 다름.
